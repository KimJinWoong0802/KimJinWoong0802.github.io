진웅이의 기초웹개발론 과제~!

IntelliJ IDEA Ultimate 설치도 하고 학생 라이센스도 받기

 

1. 먼저 IntelliJ 홈페이지 다운로드 사이트에 들어갑니다.

 

IntelliJ IDEA 다운로드하기

 

2. 왼쪽에 있는 Ultimate 버전의 다운로드 눌러 주세요.

 

 

3. 파일을 다운로드 받는동안 교육용 라이센스를 받아보아요! IntelliJ 라이선스 사이트로 들어갑니다.

 

IntelliJ 라이선스 페이지

 

4. 교육용 라이선스 관한 페이지입니다!

 

 

5. Apply Now를 눌러 주세요

 

 

6. 이메일과 이름을 비롯한 정보를 입력해주세요

 

 

7. 메일함에 온 확인 링크를 눌러주세요

 

 

8. 이후 진행하면, 자신의 라이센스 발급을 확인할 수 있습니다

 

 

9. 다운 후 설치 창입니다. Next를 눌러주세요

 

 

10. 원하는 경로를 선택하고 Next를 눌러주세요

 

 

11. Create Desktop Shortcut에 체크하고 Next를 누르세요

 

 

12. 프로그램의 시작 폴더를 정해주고 Install을 눌러주세요

 

 

13. 설치 중인 화면입니다

 

 

14. Run IntelliJ IDEA를 체크하고 Finish를 눌러주세요

 

 

15. 처음 화면에 나타난 창의 Do not import settings를 체크해주세요

 

 

16. 설치된 IntelliJ의 라이선스 설정에 들어가 인증받았던 메일과 비밀번호를 적어 Continue를 누르면 끝!!!
